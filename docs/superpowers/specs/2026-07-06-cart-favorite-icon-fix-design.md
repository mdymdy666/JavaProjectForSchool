# 购物车与收藏按钮图标修复设计

## 问题

页头购物车和商品详情收藏按钮使用了已经乱码的字符，导致按钮缺少可识别标识。功能本身不变，只修复展示与可访问性。

## 方案

- 新增轻量 `UiIcon` 组件，使用内联 SVG 提供 `cart` 和 `heart` 图标，不新增第三方依赖，也不依赖操作系统字体。
- 页头购物车按钮显示购物车图标和“购物车”文字，保留数量角标。
- 商品详情收藏按钮显示心形图标和“收藏/已收藏”文字；收藏后心形使用填充状态，卖家查看自己的商品时仍显示禁用状态。
- 两个按钮补充明确的 `aria-label`，图标本身标记为装饰内容，避免屏幕阅读器重复朗读。

## 样式

- 图标尺寸固定为 16px，使用 `currentColor` 跟随按钮文字颜色。
- 图标与文字间距 6px，按钮使用 `inline-flex` 居中，避免图标造成布局跳动。
- 不改变购物车角标、收藏请求或路由行为。

## 验证

- 组件测试验证购物车按钮包含 `cart` 图标与可读标签。
- 商品详情测试验证收藏按钮包含 `heart` 图标，并能通过收藏状态区分空心和实心。
- 运行全部前端测试和生产构建。
