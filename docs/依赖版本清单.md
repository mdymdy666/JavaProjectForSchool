# 依赖版本清单

## 背景

本清单用于记录校园二手物品交易平台项目的本机工具、项目已配置依赖、计划采用但尚未配置的依赖，以及待确认依赖。当前项目处于初始化阶段，尚未创建前端或后端工程。

## 状态说明

| 状态 | 含义 |
|---|---|
| 已检测 | 已通过命令、服务、安装目录等方式确认本机存在 |
| 已配置 | 已在项目配置文件中声明版本 |
| 未配置 | 项目尚未创建对应配置文件或尚未声明依赖版本 |
| 待确认 | 有线索但仍需用户或后续连接验证确认 |
| 未检测 | 当前命令或检查方式未发现 |
